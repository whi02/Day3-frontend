# Guestbook Front Starter

## Setup

1. Node.js 설치
   - Node.js 18 이상 버전을 사용하세요.

2. 프로젝트 의존성 설치

```bash
npm install
```

3. 개발 서버 실행

```bash
npm run dev
```

4. 빌드

```bash
npm run build
```

5. 빌드 결과 미리보기

```bash
npm run preview
```

## 주요 명령어

- `npm run dev` — 개발 서버 실행
- `npm run build` — 프로덕션 빌드 생성
- `npm run preview` — 빌드된 결과 미리보기
- `npm run lint` — ESLint 검사
